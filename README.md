# Authentication-hackathon

A full-stack application that users authentication and authorization to grant/limit users permissions to view blog posts (created during the SOC week 10 hackathon day) 

## Skills

Tools and languages:
- React
- JavaScript
- SQL
- CSS
- Node.js
- Express
- Auth0
